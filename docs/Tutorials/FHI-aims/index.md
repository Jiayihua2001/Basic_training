# FHI-aims Tutorials

[FHI-aims](https://fhi-aims.org/) is an all-electron, full-potential DFT code based on numeric atom-centered orbitals. The tutorials in this section take you from a single H₂ molecule up to 2D materials and surface adsorption.

## Quick Guide
- [Tutorial_1](./Tutorial_1/) — H₂ binding, convergence, relaxation, and simple organic molecules.
- [Tutorial_2](./Tutorial_2/) — Periodic systems: Si, Na, Fe, and Ge.
- [Tutorial_3](./Tutorial_3/) — 2D materials and surfaces: bilayer graphene and TCNQ on graphene.

> All scripts you need (`write_control.py`, `Automation.py`, `Surfaces.py`, `aimsplot.py`, and `submit.sh`) are bundled in the [Tutorial files](../../Tutorials_files.zip) and installed to `~/` by `setup_utils.sh`.
